# IELTS Study Tool

一个帮助雅思学习的React应用，包含单词记忆和写作真题练习功能。

## 功能特性

- **单词学习**：浏览雅思高频词汇，查看释义和例句（中英双语）
- **真题练习**：105道雅思写作真题，包含完整范文和高分参考
- **分类展示**：按话题、年份、题型分类浏览真题
- **关键词提示**：每道真题提供相关关键词帮助写作思路

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS

## 安装运行

```bash
npm install
npm run dev
```

## 项目结构

```
src/
├── components/       # 组件
├── data/            # 数据文件
│   ├── mockWords.ts # 单词数据
│   └── mockQuestions.ts # 真题数据（105条）
└── App.tsx         # 主应用
```

## 数据统计

- **单词数量**：800+ 雅思高频词汇
- **真题数量**：105 道写作真题（2015-2024）
- **范文质量**：Band 7-8 分标准范文

## 真题覆盖话题

- 教育类：早期教育、课程设置、留学、家庭教育、音乐教育等
- 科技类：人工智能、社交媒体、智能手机、电子书、视频会议等
- 环境类：可再生能源、塑料污染、城市绿化、野生动物保护等
- 社会类：家庭结构、博物馆、图书馆、名人责任、语言多样性等
- 工作类：公司规模、工作满意度、远程办公、团队合作、员工培训等
- 生活类：饮食、旅行、时尚、阅读、沟通等
